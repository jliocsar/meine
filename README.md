# Meine :boar:

## My personal config files for ZSH and such

This repo is a collection of my personal configuration files for ZSH and other useful tools _(a.k.a. 2 lazy 2 sync 'em)_.

### Usage :ninja:

#### Installing

Simply run the `install` script:

```bash
./install.pl
```
